什么是 ShardingSphere-on-Cloud?

ShardingSphere-on-Cloud 项目是面向 Apache ShardingSphere 的云上解决方案

通过使用 Terraform 一键部署脚本、Helm Charts、Operators、自动弹性扩容或者其它 Kubernetes 部署工具来将 Apache ShardingSphere 带至云端，并部署到 AWS、谷歌云、阿里云、CloudFormation Stack templates 的虚拟机中。

特性

一键式 Kubernetes 部署

在 Kubernetes 环境中一键部署基于 Helm Charts 的 ShardingSphere Proxy。

自动 Kubernetes DevOps

基于 Operator 的 ShardingSphere Proxy 在 Kubernetes 环境中实现一键部署和自动 DevOps。

快速 CloudFormation 部署

基于 AWS CloudFormation 的 ShardingSphere Proxy 的快速部署。

可编程的 Terraform 部署

在 AWS 环境中部署基于 Terraform 的 ShardingSphere-Proxy。

自动横向扩展

在 Kubernetes 和 AWS 上进行自定义指标的自动缩放。

负载平衡和准备

确保负载均衡器后的代理连接准备就绪

开源、标准化和一致性
